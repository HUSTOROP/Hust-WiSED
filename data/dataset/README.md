# SIB cached field

The retained 3% noise SIB concentration field is distributed separately in
[`Hust-WiSED-Dataset`](https://github.com/HUSTOROP/Hust-WiSED-Dataset) to keep
source code and experimental data distinct. Install it with:

```bash
python scripts/install_release_data.py --dataset-root ../Hust-WiSED-Dataset
```
