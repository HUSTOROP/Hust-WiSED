# SOC-EIS measurements

The public SOC-EIS measurements are distributed through the companion
[`Hust-WiSED-Dataset`](https://github.com/HUSTOROP/Hust-WiSED-Dataset)
repository. Install the B01-B11 directory tree with:

```bash
python scripts/install_release_data.py --dataset-root ../Hust-WiSED-Dataset
```

The source dataset is Mustafa *et al.*, Mendeley Data, version 2 (2024),
DOI: [10.17632/cb887gkmxw.2](https://doi.org/10.17632/cb887gkmxw.2). The
original measurements are licensed CC BY 4.0 and must be cited as described in
the companion repository.
